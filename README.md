It is an responsive security scan dashboard built with React and Tailwind CSS.


It contains three pages First one is login page which include information and login form.
Then a dasboard and ScanDetail page.

Tech Stack Used : 
 React (Vite)
 Tailwind CSS
 React Icons

 It is Mobile and desktop responsive.


 How to Run This Project on Your Laptop

Follow these steps:

git clone 
cd taskfenrir


Install Dependencies
npm install

Start Server
npm run dev
